# 簡易計算機

本專案以 React 作為核心的練習主軸，並使用 Create React App 建置。

![Alt calculator](https://i.imgur.com/2waAYj7.png)

## 開發功能

- 可對數字進行基本的四則運算(+/-/\*/÷/=/clear(c))
- 可對數字進行正負號切換
- 可對數字進行百分比計算(%)
- 可對數字進行開根號計算
- 對浮點數進行四則運算時，排除不精準的問題(ex:0.1+0.2=0.3)
- 重新整理瀏覽器可以記憶上一次輸入的數字及計算過程
- 解決無窮小數點會超出介面範圍的問題
- 超過數字上限會跳出提示
- 實作四則運算不得除以０
- 於畫面左上方顯示當時計算過程
- 於電腦版可於畫面上拖曳計算機
